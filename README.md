# 🚀 ION Gateway

Welcome to the implementation of the TonConnect protocol! Before diving in, here's where to start:


- Easily connect users to your Dapp and start blockchain interactions:
  - [ION Gateway UI React](https://www.npmjs.com/package/@ion-gateway/ui-react): Seamlessly integrate wallet connection components into your React app.
  - [ION Gateway UI](https://www.npmjs.com/package/@ion-gateway/ui): Quickly add wallet connection components to your Dapp.


- For detailed information about the TonConnect protocol, visit the [official documentation](https://docs.ton.org/develop/dapps/ion-gateway/overview).

- [📚 Latest API documentation](https://ion-gateway.github.io/sdk/)

## 📗 ION Gateway Protocol Models
- **GitHub**: [link](https://github.com/ion-gateway/sdk/tree/main/packages/protocol)
- **npm**: [link](https://www.npmjs.com/package/@ion-gateway/protocol)
- **API Documentation**: [link](https://ion-gateway.github.io/sdk/modules/_ionconnect_protocol.html)

This package contains protocol requests, responses, and event models, along with encoding and decoding functions. Use it to integrate TonConnect into your wallet app (written with TypeScript). For integrating TonConnect into your dApp, use [@ion-gateway/sdk](https://www.npmjs.com/package/@ion-gateway/sdk).

## 📙 ION Gateway UI
- **GitHub**: [link](https://github.com/ion-gateway/sdk/tree/main/packages/ui)
- **npm**: [link](https://www.npmjs.com/package/@ion-gateway/ui)
- **API Documentation**: [link](https://ion-gateway.github.io/sdk/modules/_ionconnect_ui.html)

TonConnect UI is a UI kit for TonConnect SDK. Use it to connect your app to ION wallets via the TonConnect protocol. It simplifies integration using UI elements like the "connect wallet button," "select wallet dialog," and confirmation modals.
