🚀 ION Gateway

Welcome to the implementation of the TonConnect protocol! Before diving in, here's where to start:

  • Easily connect users to your Dapp and start blockchain interactions:

    • ION Gateway UI React: Seamlessly integrate wallet connection components into your React app.
    • ION Gateway UI: Quickly add wallet connection components to your Dapp.

  • For detailed information about the TonConnect protocol, visit the official documentation.

  • 📚 Latest API documentation

📗 ION Gateway Protocol Models
  • GitHub: link
  • npm: link
  • API Documentation: link

This package contains protocol requests, responses, and event models, along with encoding and decoding functions. Use it to integrate TonConnect into your wallet app (written with TypeScript). For integrating TonConnect into your dApp, use @ion-gateway/sdk.

📙 ION Gateway UI
  • GitHub: link
  • npm: link
  • API Documentation: link

TonConnect UI is a UI kit for TonConnect SDK. Use it to connect your app to ION wallets via the TonConnect protocol. It simplifies integration using UI elements like the "connect wallet button," "select wallet dialog," and confirmation modals.

📕 ION Gateway UI React
  • GitHub: link
  • npm: link

TonConnect UI React is a React UI kit for TonConnect SDK. Use it to connect your app to ION wallets via the TonConnect protocol in React apps.
